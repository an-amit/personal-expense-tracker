Project Title:
Personal Expense Tracker
Objective
Build a simple full-stack Expense Tracker application using MERN Stack (MongoDB, Express, React, Node.js) that allows users to add, view, and delete expenses.

Requirements
Frontend (React)

Create a clean and simple UI.
Form to Add Expense (Amount, Description, Category, Date).
Display list of all expenses.
Show Total Amount spent.
Button to Delete individual expense.
Basic responsive design (optional but appreciated).

Backend (Node.js + Express)

REST APIs:
POST /api/expenses → Add new expense
GET /api/expenses → Get all expenses
DELETE /api/expenses/:id → Delete expense

Use MongoDB to store expenses.

Tech Stack: MERN
